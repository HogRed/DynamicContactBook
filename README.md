# DynamicContactBook

This project is meant to practice linked lists using pointers and manual memory management with **new** and **delete**. The project contains:

- **main.cpp**: This is the main driver file for the project. It contains the menu function that allows the user to work with the dynamic contact book.
- **contact_list.cpp**: This is the file that implements the linked list contact book. It implements the logic for all the functions declared in contact_list.h
- **contact_list.h**: This is the header file for the project. It defines the core data structures and function prototypes for a simple contact book.
